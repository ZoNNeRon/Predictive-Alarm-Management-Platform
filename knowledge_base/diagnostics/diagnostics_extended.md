# Расширенная вибродиагностика центробежных насосов (аналитическое дополнение)
# Источник: синтез на основе вибродиагностики API 610 / ISO 13373, НЕ из Таблицы 4 мануала МНХВ

> ПРИМЕЧАНИЕ: данный документ НЕ является частью Таблицы 4 руководства по эксплуатации МНХВ
> (в оригинале Таблица 4 содержит 3 неисправности). Это аналитическое дополнение,

## Диагностика по вибрационной сигнатуре

### Несоосность валов насоса и электродвигателя
- Признак: повышенная вибрация на частоте вращения 1×rpm (для 2900 об/мин ≈ 48,3 Гц), часто с осевой составляющей.
- Причина: расцентровка после монтажа или температурных деформаций.
- Действие: лазерная центровка, проверка состояния муфты (допуск несоответствия ≤ 0,05 мм по руководству МНХВ).

### Дисбаланс ротора
- Признак: доминирующая вибрация на 1×rpm в радиальном направлении.
- Причина: загрязнение, эрозия или повреждение рабочего колеса; ослабление крепления.
- Действие: осмотр и очистка рабочего колеса, динамическая балансировка по классу G2.5 (ГОСТ ИСО 1940-1).

### Износ подшипников качения
- Признак: рост вибрации на подшипниковых частотах (BPFI — внутреннее кольцо, BPFO — наружное кольцо), сопровождается ростом температуры.
- Причина: потеря смазочных свойств масла, усталостное выкрашивание.
- Действие: отбор пробы масла, замена смазки; при подтверждении износа — замена подшипников (входит в перечень КР).

### Кавитация
- Признак: широкополосная вибрация (случайный «шум») + характерный треск, падение напора и подачи.
- Причина: недостаточный кавитационный запас (NPSHa < NPSHr), засорение всасывания.
- Действие: проверка фильтра и давления на всасывании, снижение частоты вращения через ЧРП или прикрытие напорной задвижки.

### Задевание вращающихся частей (избыточное трение)
- Признак: вибрация на высоких гармониках, рост тока двигателя, рост температуры.
- Причина: уменьшение тепловых зазоров, деформация, неверная сборка.
- Действие: проверка тепловых зазоров (Таблицы 2 и 3 мануала: 0,30–0,60 мм для материалов повышенной прочности), ремонт трущихся деталей.

### Дефект механического уплотнения
- Признак: пульсирующая утечка (> 10 мл/ч) + локальная вибрация уплотнительного узла.
- Причина: радиальные царапины на торцевых кольцах, износ пары трения.
- Действие: при неглубоких царапинах — шлифовка; при глубоких — замена пары трения или уплотнения целиком.

## Пороговая логика реагирования по вибрации

| Уровень виброскорости | Состояние | Действие |
|---|---|---|
| < 3,0 мм/с СКЗ | Норма | Штатный мониторинг |
| ≥ 3,0 мм/с СКЗ | Warning | Диагностика причины по сигнатуре, усиленный контроль |
| ≥ 8,0 мм/с (пиковое) | Critical | Немедленный останов, дефектация (руководство МНХВ, п. 3.5.2) |
